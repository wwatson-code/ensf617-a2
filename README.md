# ensf617-a2

Garbage classification multi-modal model 

To make an inference, please make sure you install all of the libraries in requirements.txt

After installing, you are good to run our code


The 'final_predictions' notebook contains the code to load, and run the model to obtain the final predictions and find the confusion matrix 

The 'final_trainer' notebook contains the code to train the model and save a new model
